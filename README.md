## 现有功能

* 支持根据特定目标爬取图片(目前仅支持百度贴吧)
* 判断图片是否为微信群二维码及是否尚在可扫描有效期内
* 浏览爬取结果

## 使用方法

### 设置爬取目标

配置文件：/Data/tieba_spider_key.yaml

```yaml
- Key: "王者荣耀微信群"
  Page: 1
```

依此格式添加爬取贴吧，Key为贴吧名称，Page为爬取深度。如Page设置为1，则仅爬取贴吧首页，可根据贴吧活跃程度及爬取价值自行调整深度。
